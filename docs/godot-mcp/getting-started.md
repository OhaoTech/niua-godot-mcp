# Getting started

**Use the [README](../../README.md).** That’s the full install + first game guide.

Short version:

1. Paste the install prompt from the README into your AI, **or** clone + one setup command.  
2. Restart the AI client.  
3. Ask: `Call get_godot_version`.  
4. Ask it to create a project and build a small 3D scene.

Stuck? [troubleshooting.md](troubleshooting.md)
